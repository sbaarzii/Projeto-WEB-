Bem-vindo ao Sistema de Biblioteca

Utilize o menu de navegação acima para gerenciar os principais cadastros do sistema:

  • Autores: Cadastre, edite e consulte os autores disponíveis no sistema.
  • Gêneros: Registre os gêneros por categorias como Romance, Ficção, Fantasia, etc.
  • Livros: Registre os livros, vinculando-os a autores e gêneros já cadastrados.

Essa dashboard é o ponto de partida para acessar e administrar todas as informações da biblioteca de forma simples e prática.
